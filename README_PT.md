## O Que É

`department.skill` ajuda você a entender como um time realmente funciona: sinais implícitos, fluxo de ownership, regras de onboarding e respostas mais seguras.

## Fontes Compatíveis

- Conversas de WeChat / Feishu / DingTalk
- Grupos de equipe e projeto
- Documentos, PDF, Markdown e notas em texto
- Reviews de código, PRs e issues
- Contexto adicional fornecido pelo usuário
