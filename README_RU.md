## Что Это

`department.skill` помогает понять, как команда реально работает: скрытые сигналы, движение ownership, правила онбординга и более безопасные ответы.

## Поддерживаемые Источники

- Чаты WeChat / Feishu / DingTalk
- Командные и проектные группы
- Документы, PDF, Markdown и текстовые заметки
- Code review, PR и issue комментарии
- Дополнительный контекст от пользователя
