## Qué Es

`department.skill` te ayuda a entender cómo funciona realmente un equipo: señales implícitas, flujo de ownership, reglas de onboarding y respuestas más seguras.

## Fuentes Compatibles

- Chats de WeChat / Feishu / DingTalk
- Grupos de equipo y proyecto
- Documentos, PDF, Markdown y notas de texto
- Reviews de código, PR e issues
- Contexto adicional proporcionado por el usuario

## Instalación

Consulta `INSTALL.md`.

Herramientas opcionales:

```bash
pip3 install -r requirements.txt
playwright install chromium
```

## Herramientas Opcionales

Incluye herramientas Python para Feishu, DingTalk, Slack, email, WeChat, QQ y exportaciones genéricas de chat. Consulta `tools/README.md` y `docs/THIRD_PARTY.md`.

## Uso

```text
/department
```

## Ejemplos

- Onboarding: quién es accesible y qué se puede preguntar directo
- Subtexto del jefe: qué significa realmente revísalo otra vez
- Mapa del equipo: quién impulsa, quién decide y quién difumina el ownership
