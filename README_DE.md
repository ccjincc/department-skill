## Was Es Ist

`department.skill` hilft dir zu verstehen, wie ein Team wirklich funktioniert: versteckte Signale, Ownership-Fluss, Onboarding-Regeln und sicherere Reaktionsstrategien.

## Unterstützte Datenquellen

- WeChat / Feishu / DingTalk Chats
- Team- und Projektgruppen
- Dokumente, PDF, Markdown und Textnotizen
- Code-Reviews, PR-Kommentare und Issues
- Zusätzlicher Beziehungskontext vom Nutzer

## Installation
